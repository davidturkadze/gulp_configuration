1.  Download Node.js
2.  Install gulp globally: npm i gulp-cli -g
3.  Create project folder
4.  cd to folder: npm init (entry point maybe index.js)
5.  npm i gulp --save-dev
6.  npm i gulp-sass --save-dev
7.  npm i browser-sync--save-dev
8.  npm i normalize.css (not necessary)
9.  npm i gulp-uglify gulp-concat --save-dev
10. npm i gulp-rename --save-dev

If you want to use this configuration in another projects, just copy only app, gulpfile.js and package.json in your new folder and tipp in the console (in that folder) npm i.
